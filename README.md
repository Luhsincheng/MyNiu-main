# 🎮 MyNIU — 我的宜蘭大學

> 一款以**國立宜蘭大學**校園生活為主題的回合制角色扮演 (RPG) 桌面遊戲。

---

## 📖 專案簡介

**MyNIU（我的宜蘭大學）** 是一款 Windows 桌面 RPG 遊戲，玩家扮演大學生角色，在校園中透過日常活動（汁妹、讀書、玩社團）累積經驗值與提升角色數值，最終挑戰以校園人物為原型的六大 Boss，體驗趣味橫生的大學冒險！

---

## ✨ 遊戲特色

- 🧑‍🤝‍🧑 **4 位可選角色**：帥隆、帥廷（帥郭）、帥誠、帥泓，每位角色擁有不同的初始數值與專屬技能
- 👾 **6 大 Boss 關卡**：微笑殺手、正妹汶涓老師、無信德、Python 小彭、心碎小子、武淵堂堂主
- ⚔️ **回合制自動戰鬥系統**：基於攻擊力、防禦力、智商等數值的戰鬥計算，含閃避機制
- 📈 **角色成長系統**：透過日常活動獲取經驗值並升級，提升血量、攻擊、防禦、智商
- 🎲 **快篩小遊戲**：開局隨機快篩，陽性直接結束、陰性才能繼續遊戲
- 🖼️ **圖形化介面**：使用 Windows Forms 打造的 GUI 操作介面

## 🎮 遊戲玩法

### 1. 開始遊戲
啟動後會看到「我的宜蘭大學」標題畫面，點擊畫面中央的圖片進行**快篩**：
- **陰性** → 進入角色選擇畫面
- **陽性** → 遊戲結束 😷

### 2. 選擇角色
從 4 位角色中選擇一位，各角色初始數值如下：

| 角色 | 血量 (HP) | 攻擊力 (ATK) | 防禦力 (DEF) | 智商 (IQ) |
|------|-----------|--------------|--------------|-----------|
| 帥隆 | 1000 | 50 | 40 | 50 |
| 帥廷 | 1000 | 50 | 50 | 40 |
| 帥誠 | 1000 | 30 | 40 | 50 |
| 帥泓 | 1000 | 50 | 60 | 50 |

### 3. 日常活動（養成階段）
在角色畫面中可進行以下活動來獲取經驗值：

| 活動 | 經驗值 | 說明 |
|------|--------|------|
| 汁妹 | +20 EXP | 社交活動 |
| 玩社團 | +17 EXP | 課外活動 |
| 讀書 | +15 EXP | 學業研習 |

升級時（經驗值達到 `等級 × 20`）角色數值將提升：血量 +30、攻擊力 +10、防禦力 +10、智商 +10。

### 4. Boss 戰鬥
切換至 Boss 畫面後可挑戰 Boss，戰鬥為**自動回合制**：
- 雙方依智商決定攻防順序
- 智商較高者優先攻擊
- 含閃避判定機制
- 擊敗當前 Boss 後自動進入下一關

### 5. 通關
依序擊敗所有 6 位 Boss 即可通關！

## 🚀 如何執行

### 前置需求
- **Windows 10** 或以上
- **Visual Studio 2022**（含「使用 C++ 的桌面開發」與「.NET 桌面開發」工作負載）
- **.NET Framework 4.8**

### 建置與執行步驟

1. **克隆專案**
   ```bash
   git clone https://github.com/<your-username>/MyNiu.git
   ```

2. **開啟方案**
   - 使用 Visual Studio 2022 開啟 `MyNIU/MyNIU/MyNIU.sln`

3. **設定組態**
   - 將平台設為 **x64**
   - 將組態設為 **Debug** 或 **Release**

4. **建置並執行**
   - 按下 `Ctrl + F5` 建置並執行
   - 確保 `image/` 資料夾與執行檔位於同一目錄下
